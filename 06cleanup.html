资源清理

资源清理

数字资产盘活机器人中使用的服务绝大多数都是On Demand的类型，它们仅仅在运行时启动并产生费用。只有推理终端节点在使用后会一直运行，如果您不需要频繁启动数字资产盘活任务，您也可以发送REST请求或从控制台中关闭推理终端节点。

针对不同的机器人，请从Terminal中输入相应的命令来清理机器人所使用的资源：

情感分析机器人

curl -d '{"bot_name": "SENTIMENT_ANALYSIS"}' -X DELETE 'https://xxxxxxxxxx.execute-api.cn-northwest-1.amazonaws.com.cn/Prod/create/proxy‘

车型分类机器人

curl -d '{"bot_name": "CAR_ACCIDENT_INSPECTOR"}' -X DELETE 'https://xxxxxxxxxx.execute-api.cn-northwest-1.amazonaws.com.cn/Prod/create/proxy‘

身份证识别机器人

curl -d '{"bot_name": "CHINESE_ID_OCR"}' -X DELETE 'https://xxxxxxxxxx.execute-api.cn-northwest-1.amazonaws.com.cn/Prod/create/proxy‘
